# 콘솔 오류 숨김 해제 가이드

크롬 브라우저 개발자 도구에서 "47개 숨김" 메시지를 모두 표시하는 방법:

## 방법 1: 콘솔 필터 설정

1. **개발자 도구 열기** (F12 또는 Ctrl+Shift+I)
2. **Console 탭** 클릭
3. **필터 아이콘** (필터 입력창 옆) 클릭
4. 다음 옵션들을 확인/해제:
   - ✅ **Verbose** 체크 (모든 로그 레벨 표시)
   - ✅ **Info** 체크
   - ✅ **Warnings** 체크
   - ✅ **Errors** 체크
   - ❌ **Hide network messages** 체크 해제 (네트워크 메시지도 표시)

## 방법 2: 콘솔 설정 메뉴

1. **개발자 도구 열기**
2. **Console 탭**에서 **설정 아이콘** (톱니바퀴) 클릭
3. 다음 옵션 확인:
   - ✅ **Show timestamps** (타임스탬프 표시)
   - ✅ **Show CORS errors in console** (CORS 오류 표시)
   - ❌ **Group similar** 체크 해제 (유사한 메시지 그룹화 해제)
   - ❌ **Hide network messages** 체크 해제

## 방법 3: 콘솔 필터 직접 입력

콘솔 필터 입력창에 다음을 입력:
```
-level:verbose -level:info -level:warning -level:error
```
또는 모든 메시지를 보려면 필터를 비워두세요.

## 방법 4: 콘솔 컨텍스트 메뉴

1. 콘솔에서 **우클릭**
2. **"Show all messages"** 또는 **"Show verbose messages"** 선택

## 참고

- "숨김" 메시지는 주로 중복되거나 유사한 메시지가 많을 때 브라우저가 자동으로 그룹화한 것입니다.
- 개발 중에는 모든 오류를 확인하는 것이 중요하므로 위 설정을 적용하세요.
- 프로덕션 환경에서는 성능을 위해 일부 메시지를 숨기는 것이 좋을 수 있습니다.
