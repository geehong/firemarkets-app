# 키워드 분석 및 불용어 자동 업데이트 가이드

`analyze_keywords.py`는 금융 뉴스 데이터의 노이즈를 제거하고 대시보드의 키워드 품질을 최상으로 유지하기 위한 통계 분석 도구입니다.

## 🎯 주요 목적
- **소음(Noise) 식별**: 뉴스 제목에서 반복적으로 나타나지만 정보 가치가 낮은 일반 단어(상용구)를 찾아냅니다.
- **필터 자동 업데이트**: 분석 결과, 특정 임계값(8%)을 넘는 새로운 단어가 발견되면 `backend/app/utils/stop_words.json` 파일에 자동으로 병합하여 저장합니다.
- **대시보드 최적화**: 업데이트된 불용어 리스트는 `dashboard.py`에 즉시 반영되어 더 정밀한 키워드 추출을 가능하게 합니다.

## 🚀 실행 방법 (Docker 환경)

터미널에서 아래 명령어를 실행하면 전수 조사가 시작되고 불용어 리스트가 자동으로 갱신됩니다.

```bash
docker exec fire_markets_backend python scripts/analyze_keywords.py
```

## 🧠 작동 원리
1. **전수 조사**: 데이터베이스 내 모든 상태(`Draft`, `Published`, `Private` 등)의 기사 제목을 스캔합니다.
2. **비율 계산**: 각 단어가 전체 기사 중 몇 %의 기사에서 나타나는지(`Doc Ratio`) 계산합니다.
3. **임계값 적용**: `Doc Ratio`가 **8%**를 초과하는 단어는 '금융 뉴스 상용구'로 간주합니다.
4. **자동 병합**: 추출된 단어 중 기존 `stop_words.json`에 없는 단어만 골라내어 중복 없이 추가하고, 가나다/알파벳 순으로 정렬하여 저장합니다.

## 💡 관리 팁
- **정기 실행**: 데이터가 수천 건 단위로 쌓이거나, 새로운 유행어가 뉴스 제목에 고착화될 때 한 번씩 실행해주시면 좋습니다.
- **위젯 모니터링**: 대시보드의 'Hot Draft Keywords'에 다시 일반적인 단어(예: '전략', '상승')가 보이기 시작하면 이 스크립트를 실행하여 필터를 최신화하십시오.
- **수동 관리**: 특정 단어를 강제로 필터링하거나 해제하고 싶다면 `backend/app/utils/stop_words.json` 파일을 직접 수정하여 즉시 반영할 수 있습니다.
